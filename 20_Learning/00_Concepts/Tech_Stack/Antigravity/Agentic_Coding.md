# Agentic Coding

## Definition
**Agentic Coding**은 개발자가 직접 코드를 작성하는 대신, 고수준의 계획(Plan)과 의도를 제공하고 AI 에이전트가 이를 실행(Execution)하는 협업 방법론입니다.

## Core Philosophy
1.  **One Task, One Workflow**: 복잡한 작업을 관리 가능한 단위의 워크플로우로 분할합니다.
2.  **Context is King**: 에이전트에게 충분한 맥락 정보를 제공하여 환각(Hallucination)을 줄입니다.
3.  **Human in the Loop**: 중요한 의사결정 시점에는 반드시 사용자의 승인을 거칩니다.

## vs Traditional Coding
단순한 코드 자동 완성이 아니라, 전체 소프트웨어 개발 수명 주기(SDLC)의 프로세스를 에이전트에게 위임하는 것이 특징입니다.
