# Feature Engineering (특성 공학)

**Category**: #concept/ml/preprocessing
**Source**: [[24_ML_Preprocessing_Encoding_Scaling]]

## 정의
원시 데이터를 머신러닝 모델이 더 잘 학습할 수 있는 형태로 변환하는 과정입니다.

## 주요 기법
- **Feature Selection**: 중요한 특성만 선택
- **Feature Extraction**: 새로운 특성 생성 (PCA 등)
- **Feature Transformation**: 스케일링, 인코딩
- **Feature Creation**: 도메인 지식 기반 새 특성 생성

## 중요성
"데이터와 특성이 알고리즘보다 중요하다" - 좋은 특성이 모델 성능을 크게 향상시킵니다.
