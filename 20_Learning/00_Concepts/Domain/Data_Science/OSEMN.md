# OSEMN

## Definition
**OSEMN**은 데이터 과학 프로젝트의 표준 수명 주기를 나타내는 두문자어(Acronym)입니다.

## 5 Stages
1.  **Obtain (획득)**: 다양한 소스에서 데이터를 수집합니다.
2.  **Scrub (세척)**: 데이터를 정제하고 전처리합니다 (Cleaning).
3.  **Explore (탐색)**: EDA를 통해 데이터의 패턴과 특징을 파악합니다.
4.  **Model (모델링)**: 예측 모델을 구축하거나 통계적 분석을 수행합니다.
5.  **Interpret (해석)**: 결과를 해석하고 비즈니스 인사이트로 변환하여 보고합니다.
