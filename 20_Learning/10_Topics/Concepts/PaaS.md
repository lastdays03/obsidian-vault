# PaaS (Platform as a Service)

## Definition
**PaaS**는 개발자가 인프라(OS, 네트워크, 스토리지)를 신경 쓰지 않고 애플리케이션을 개발, 실행, 관리할 수 있는 플랫폼을 제공하는 클라우드 모델입니다. 2026년에는 **AI PaaS**와 **Frontend Cloud**(Vercel)가 주류입니다.

## Key Features
- **Abstraction**: 서버 관리 복잡성을 추상화하여 개발 생산성을 높입니다.
- **Managed Services**: 데이터베이스, 메시징 큐, 캐시 등을 관리형 서비스로 제공합니다.
- **Scalability**: 트래픽에 따른 자동 확장을 플랫폼이 담당합니다.

## Examples
- **Vercel**: 프론트엔드 특화 PaaS.
- **Supabase**: Backend-as-a-Service (PaaS의 일종).
- **Heroku**: 고전적인 PaaS의 시초.
