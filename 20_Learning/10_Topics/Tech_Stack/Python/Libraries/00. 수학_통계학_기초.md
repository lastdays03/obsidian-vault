## 1. 기본 수학 개념

### 1.1 산술 연산

**덧셈, 뺄셈, 곱셈, 나눗셈**
- 기본 사칙연산은 모든 프로그래밍 언어와 동일
- NumPy/Pandas에서는 **벡터화 연산**으로 배열 전체에 한 번에 적용

**거듭제곱 (Power)**
- a² = a × a
- aⁿ = a를 n번 곱한 값
- 제곱근: √a = a^(1/2)

**절댓값 (Absolute Value)**
- |a| = a (a ≥ 0일 때)
- |a| = -a (a < 0일 때)
- 예: |5| = 5, |-5| = 5

### 1.2 지수와 로그

**지수 함수 (Exponential)**
- e^x: 자연상수 e(≈2.718)의 x제곱
- 성장률, 감쇠율 계산에 사용
- 예: e^1 ≈ 2.718, e^0 = 1

**로그 함수 (Logarithm)**
- log_a(x): a를 밑으로 하는 x의 로그
- log_e(x) = ln(x): 자연로그
- log_10(x): 상용로그
- 성질:
  - log(ab) = log(a) + log(b)
  - log(a/b) = log(a) - log(b)
  - log(a^b) = b·log(a)

**사용 사례**
- 데이터 정규화 (로그 변환)
- 지수적 증가/감소 데이터 처리
- 신호 처리

### 1.3 삼각함수

**기본 삼각함수**
- sin(θ): 사인 함수
- cos(θ): 코사인 함수
- tan(θ): 탄젠트 함수 = sin(θ)/cos(θ)

**각도 단위**
- 도(degree): 360도 = 한 바퀴
- 라디안(radian): 2π = 한 바퀴
- 변환: 라디안 = 도 × (π/180)

**사용 사례**
- 신호 처리 (주기적 패턴)
- 회전 변환
- 파동 분석

## 2. 선형대수 (Linear Algebra)

### 2.1 벡터 (Vector)

**정의**
- 크기와 방향을 가진 수학적 객체
- 1차원 배열로 표현: [x₁, x₂, ..., xₙ]

**벡터 연산**
- **덧셈**: [a₁, a₂] + [b₁, b₂] = [a₁+b₁, a₂+b₂]
- **스칼라 곱**: k × [a₁, a₂] = [k·a₁, k·a₂]
- **내적 (Dot Product)**: 
  - a · b = a₁b₁ + a₂b₂ + ... + aₙbₙ
  - 결과는 스칼라(숫자)
  - 두 벡터의 유사도 측정에 사용

**벡터의 크기 (Norm)**
- ||a|| = √(a₁² + a₂² + ... + aₙ²)
- 유클리드 거리

### 2.2 행렬 (Matrix)

**정의**
- 숫자를 직사각형 형태로 배열한 것
- m × n 행렬: m개 행, n개 열
- 2차원 배열로 표현

```
A = [a₁₁  a₁₂  a₁₃]
    [a₂₁  a₂₂  a₂₃]
```

**행렬 연산**

**덧셈/뺄셈**
- 같은 크기의 행렬만 가능
- 각 원소별로 연산

**스칼라 곱**
- 모든 원소에 스칼라를 곱함

**행렬 곱셈**
- A (m×n) × B (n×p) = C (m×p)
- 첫 번째 행렬의 열 수 = 두 번째 행렬의 행 수
- C의 (i,j) 원소 = A의 i번째 행 · B의 j번째 열 (내적)

**전치 행렬 (Transpose)**
- 행과 열을 바꾼 행렬
- A^T: A의 전치
- (A^T)^T = A

**단위 행렬 (Identity Matrix)**
- 대각선 원소가 모두 1, 나머지는 0
- I × A = A × I = A (곱셈의 항등원)

**역행렬 (Inverse Matrix)**
- A × A^(-1) = I
- 정사각 행렬만 역행렬 존재 가능
- 행렬식(determinant)이 0이 아니어야 함

**행렬식 (Determinant)**
- 2×2 행렬: det(A) = a₁₁a₂₂ - a₁₂a₂₁
- 행렬의 특성을 나타내는 스칼라 값
- det(A) = 0이면 역행렬 없음

### 2.3 고유값과 고유벡터

**정의**
- A × v = λ × v
- v: 고유벡터 (eigenvector)
- λ: 고유값 (eigenvalue)

**의미**
- 행렬 A를 곱해도 방향이 변하지 않는 벡터
- 주성분 분석(PCA)에 사용

**사용 사례**
- 차원 축소
- 데이터 압축
- 주요 방향 찾기

## 3. 기술통계 (Descriptive Statistics)

### 3.1 중심 경향 측도 (Measures of Central Tendency)

**평균 (Mean)**
- 산술 평균: 모든 값의 합을 개수로 나눈 값
- 공식: x̄ = (x₁ + x₂ + ... + xₙ) / n
- 예: [1, 2, 3, 4, 5]의 평균 = 15/5 = 3
- **특징**: 이상치에 민감

**중앙값 (Median)**
- 데이터를 정렬했을 때 중앙에 위치한 값
- 홀수 개: (n+1)/2 번째 값
- 짝수 개: n/2 번째와 (n/2)+1 번째 값의 평균
- 예: [1, 2, 3, 4, 5]의 중앙값 = 3
- **특징**: 이상치에 덜 민감

**최빈값 (Mode)**
- 가장 자주 나타나는 값
- 범주형 데이터에 유용

**언제 사용할까?**
- **평균**: 정규 분포, 이상치 적을 때
- **중앙값**: 이상치 많을 때, 비대칭 분포
- **최빈값**: 범주형 데이터, 최빈 패턴 찾기

### 3.2 산포 측도 (Measures of Dispersion)

**분산 (Variance)**
- 데이터가 평균으로부터 얼마나 퍼져있는지
- 공식: Var(X) = Σ(xᵢ - x̄)² / (n-1)
- 표본 분산: n-1로 나눔 (비편향 추정량)
- 모집단 분산: n으로 나눔

**표준편차 (Standard Deviation)**
- 분산의 제곱근
- 공식: σ = √Var(X)
- 평균과 같은 단위를 가짐
- 해석이 더 쉬움

**범위 (Range)**
- 최댓값 - 최솟값
- 가장 간단한 산포 측도

**사분위수 범위 (IQR: Interquartile Range)**
- Q₃ - Q₁ (75% 백분위수 - 25% 백분위수)
- 중앙 50% 데이터의 범위
- 이상치 탐지에 사용

**예제**
```
데이터: [1, 2, 3, 4, 5]
평균: 3
분산: ((1-3)² + (2-3)² + (3-3)² + (4-3)² + (5-3)²) / 4 = 2.5
표준편차: √2.5 ≈ 1.58
```

### 3.3 백분위수와 사분위수

**백분위수 (Percentile)**
- 데이터를 정렬했을 때 특정 비율의 값
- 예: 75% 백분위수 = 상위 25%의 시작점

**사분위수 (Quartile)**
- Q₁ (1사분위수): 25% 백분위수
- Q₂ (2사분위수): 50% 백분위수 = 중앙값
- Q₃ (3사분위수): 75% 백분위수

**박스 플롯 구성**
- 최솟값, Q₁, 중앙값, Q₃, 최댓값
- IQR = Q₃ - Q₁
- 이상치: Q₁ - 1.5×IQR 미만 또는 Q₃ + 1.5×IQR 초과

### 3.4 상관관계와 공분산

**공분산 (Covariance)**
- 두 변수가 함께 변하는 정도
- 공식: Cov(X,Y) = Σ(xᵢ - x̄)(yᵢ - ȳ) / (n-1)
- 양수: 양의 상관관계 (한쪽 증가 시 다른 쪽도 증가)
- 음수: 음의 상관관계 (한쪽 증가 시 다른 쪽 감소)
- 0: 상관관계 없음

**상관계수 (Correlation Coefficient)**
- 공분산을 표준화한 값
- 공식: r = Cov(X,Y) / (σₓ × σᵧ)
- 범위: -1 ≤ r ≤ 1
- r = 1: 완전한 양의 상관관계
- r = -1: 완전한 음의 상관관계
- r = 0: 상관관계 없음

**해석**
- |r| > 0.7: 강한 상관관계
- 0.3 < |r| ≤ 0.7: 중간 상관관계
- |r| ≤ 0.3: 약한 상관관계

**주의사항**
- 상관관계 ≠ 인과관계
- 비선형 관계는 감지 못할 수 있음

## 4. 확률론 (Probability Theory)

### 4.1 기본 개념

**확률 (Probability)**
- 사건이 일어날 가능성
- 0 ≤ P(A) ≤ 1
- P(A) = 1: 반드시 일어남
- P(A) = 0: 절대 일어나지 않음

**확률 분포 (Probability Distribution)**
- 확률 변수가 가질 수 있는 값과 그 확률
- 이산형: 특정 값만 가짐 (주사위 눈)
- 연속형: 연속적인 값 (키, 몸무게)

### 4.2 주요 확률 분포

**정규 분포 (Normal Distribution / Gaussian Distribution)**
- 종 모양의 대칭 분포
- 평균(μ)과 표준편차(σ)로 정의
- 표기: N(μ, σ²)
- **특징**:
  - 평균 주변에 대부분의 값이 집중
  - 68-95-99.7 규칙:
    - 평균 ± 1σ: 약 68%
    - 평균 ± 2σ: 약 95%
    - 평균 ± 3σ: 약 99.7%

**표준 정규 분포**
- 평균 0, 표준편차 1인 정규 분포
- N(0, 1)
- 모든 정규 분포는 표준 정규 분포로 변환 가능

**균등 분포 (Uniform Distribution)**
- 모든 값이 동일한 확률
- 예: 주사위 (각 눈이 1/6 확률)

**이항 분포 (Binomial Distribution)**
- n번 시도 중 k번 성공할 확률
- 예: 동전 10번 던져서 앞면 7번 나올 확률
- 파라미터: n (시도 횟수), p (성공 확률)

**포아송 분포 (Poisson Distribution)**
- 희귀한 사건이 일정 시간 내 발생할 횟수
- 예: 하루에 받는 이메일 수
- 파라미터: λ (평균 발생 횟수)

**지수 분포 (Exponential Distribution)**
- 사건 사이의 대기 시간
- 예: 고객 도착 간격
- 파라미터: λ (비율)

### 4.3 난수 생성

**의사 난수 (Pseudo-Random Number)**
- 컴퓨터로 생성하는 난수
- 알고리즘 기반 (완전한 난수 아님)
- 시드(seed)로 재현 가능

**시드 (Seed)**
- 난수 생성기의 초기값
- 같은 시드 → 같은 난수 시퀀스
- 실험 재현성 보장에 중요

## 5. 데이터 처리 개념

### 5.1 벡터화 (Vectorization)

**정의**
- 반복문 대신 배열 전체에 연산을 한 번에 적용
- NumPy/Pandas의 핵심 개념

**예제**
```python
# 비효율적 (반복문)
result = []
for x in data:
    result.append(x * 2)

# 효율적 (벡터화)
result = data * 2
```

**장점**
- 빠른 속도 (C로 구현)
- 간결한 코드
- 메모리 효율적

### 5.2 브로드캐스팅 (Broadcasting)

**정의**
- 서로 다른 크기의 배열 간 연산
- 작은 배열을 큰 배열의 형태로 자동 확장

**규칙**
1. 차원 수가 다르면 작은 쪽 앞에 1 추가
2. 각 차원의 크기가 같거나 하나가 1이어야 함
3. 크기가 1인 차원은 확장

**예제**
```
[1, 2, 3] + 10  →  [11, 12, 13]
(1×3)     (스칼라)

[[1, 2],    +  [10, 20]  →  [[11, 22],
 [3, 4]]                      [13, 24]]
(2×2)      (1×2)            (2×2)
```

### 5.3 차원 (Dimension)

**1차원 (1D)**
- 벡터, 리스트
- 예: [1, 2, 3, 4, 5]

**2차원 (2D)**
- 행렬, 테이블
- 예: [[1, 2, 3],
       [4, 5, 6]]

**3차원 이상 (3D+)**
- 텐서
- 예: 이미지 (높이 × 너비 × 채널)
- 예: 비디오 (시간 × 높이 × 너비 × 채널)

**형태 (Shape)**
- 배열의 각 차원의 크기
- 예: (3, 4) = 3행 4열

### 5.4 인덱싱과 슬라이싱

**인덱싱 (Indexing)**
- 특정 위치의 원소 접근
- 0부터 시작 (Python 스타일)

**슬라이싱 (Slicing)**
- 범위의 원소 선택
- [시작:끝:간격]
- 끝 인덱스는 포함 안 됨 (Python 스타일)

**예제**
```python
arr = [0, 1, 2, 3, 4, 5]
arr[0]      # 0 (첫 번째)
arr[1:4]    # [1, 2, 3] (1~3번째)
arr[::2]    # [0, 2, 4] (2칸씩)
arr[::-1]   # [5, 4, 3, 2, 1, 0] (역순)
```

### 5.5 정규화 (Normalization)

**Z-score 정규화 (Standardization)**
- 평균 0, 표준편차 1로 변환
- 공식: z = (x - μ) / σ
- 정규 분포 데이터에 적합

**Min-Max 정규화**
- 0~1 범위로 변환
- 공식: x' = (x - min) / (max - min)
- 모든 데이터를 같은 스케일로

**사용 사례**
- 머신러닝 전처리
- 서로 다른 스케일의 데이터 비교
- 수치적 안정성 향상

## 6. 수치 계산 기초

### 6.1 부동소수점 (Floating Point)

**정의**
- 실수를 근사적으로 표현
- 정확하지 않을 수 있음

**주의사항**
```python
0.1 + 0.2  # 0.30000000000000004 (정확히 0.3 아님)
```

**해결 방법**
- 작은 오차는 무시
- 반올림 사용
- 정확한 계산이 필요하면 Decimal 사용

### 6.2 결측치 (Missing Values)

**NaN (Not a Number)**
- 숫자가 아닌 값
- 계산 결과가 정의되지 않을 때
- 예: 0/0, √(-1)

**None**
- 값이 없음을 나타냄
- Python의 None 객체

**처리 방법**
- 제거: dropna()
- 채우기: fillna()
- 보간: interpolate()

### 6.3 수치 안정성 (Numerical Stability)

**오버플로우 (Overflow)**
- 값이 너무 커서 표현 불가
- 예: e^1000

**언더플로우 (Underflow)**
- 값이 너무 작아서 0으로 처리
- 예: e^(-1000)

**해결 방법**
- 로그 공간에서 계산
- 적절한 스케일링
- 안정적인 알고리즘 사용

### 6.4 수치 미분과 적분

**수치 미분 (Numerical Differentiation)**
- 도함수의 근사값 계산
- 공식: f'(x) ≈ (f(x+h) - f(x)) / h
- h는 작은 값 (예: 1e-5)

**수치 적분 (Numerical Integration)**
- 적분의 근사값 계산
- 사다리꼴 법칙, 심슨 법칙 등

**사용 사례**
- 최적화 (경사 하강법)
- 미분 방정식 풀이
- 면적/부피 계산

## 7. 실전 활용 팁

### 7.1 통계 해석 가이드

**평균 vs 중앙값**
- 대칭 분포: 평균 ≈ 중앙값
- 비대칭 분포: 평균 ≠ 중앙값
- 이상치 많을 때: 중앙값 사용 권장

**표준편차 해석**
- 작은 표준편차: 데이터가 평균 주변에 집중
- 큰 표준편차: 데이터가 넓게 퍼져있음

**상관계수 해석**
- |r| > 0.7: 강한 관계 (주의 깊게 확인)
- 0.3 < |r| ≤ 0.7: 중간 관계
- |r| ≤ 0.3: 약한 관계

### 7.2 데이터 전처리 체크리스트

1. **결측치 확인 및 처리**
   - isna().sum()으로 확인
   - 제거 또는 채우기 결정

2. **이상치 탐지**
   - IQR 방법: Q₁ - 1.5×IQR, Q₃ + 1.5×IQR
   - Z-score 방법: |z| > 3

3. **정규화**
   - 서로 다른 스케일의 변수 정규화
   - 머신러닝 전 필수

4. **분포 확인**
   - 히스토그램으로 분포 확인
   - 정규 분포 여부 확인

### 7.3 일반적인 실수와 해결

**실수 1: 평균만 보고 판단**
- 해결: 중앙값, 표준편차도 함께 확인

**실수 2: 상관관계 = 인과관계**
- 해결: 인과관계는 별도 분석 필요

**실수 3: 이상치 무조건 제거**
- 해결: 이상치의 의미 파악 후 결정

**실수 4: 정규화 안 함**
- 해결: 스케일이 다른 변수는 정규화

**실수 5: 결측치 무시**
- 해결: 결측치 패턴 분석 후 처리

## 8. 공식 요약

### 8.1 통계 공식

**평균**
```
x̄ = (Σxᵢ) / n
```

**분산**
```
Var(X) = Σ(xᵢ - x̄)² / (n-1)
```

**표준편차**
```
σ = √Var(X)
```

**Z-score**
```
z = (x - μ) / σ
```

**상관계수**
```
r = Cov(X,Y) / (σₓ × σᵧ)
```

### 8.2 행렬 공식

**행렬 곱셈**
```
C[i,j] = Σ A[i,k] × B[k,j]
```

**전치 행렬**
```
(A^T)[i,j] = A[j,i]
```

**2×2 행렬식**
```
det(A) = a₁₁a₂₂ - a₁₂a₂₁
```
