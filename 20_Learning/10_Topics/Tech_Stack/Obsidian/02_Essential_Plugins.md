# 02. Essential Plugins (필수 플러그인)

**Goal**: 생산성을 높여주는 필수 플러그인 3대장 활용법.

## 1. Calendar (캘린더)
- **위치**: 우측 사이드바 상단.
- **기능**:
    - 날짜를 클릭하여 해당 날짜의 **Daily Note**를 즉시 생성하거나 이동합니다.
    - 글을 많이 쓴 날은 점(Dot)이나 색상으로 시각화됩니다 (GitHub 스타일).
- **실습**: 오늘 날짜를 클릭하여 Daily Note를 만들어 보세요.

## 2. Recent Files (최근 파일)
- **위치**: 좌측 사이드바 (아이콘: 시계 모양).
- **기능**: 방금 닫은 파일을 다시 찾을 때 유용합니다.
- **팁**: `Cmd + P` -> `Recent Files: Focus on most recent file` 명령어로 바로 전 파일로 이동할 수 있습니다.

## 3. Emoji Shortcodes (이모지 단축코드)
- **기능**: 슬랙이나 디스코드처럼 콜론(`:`)으로 이모지를 입력합니다.
- **문법**: `:fire:` -> 🔥, `:check:` -> ✅
