# 01. Shortcut Mastery (단축키 마스터)

**Goal**: 마우스 없이 옵시디언을 자유자재로 다루기.

## 1. 커맨드 팔레트 (`Cmd + P`)
모든 기능의 입구입니다. 단축키를 다 외울 필요 없이 이 키 하나만 기억하면 됩니다.
- **실습**:
    1. `Cmd + P`를 누릅니다.
    2. `Split`을 검색하여 "Split right"를 실행해 보세요.
    3. 다시 `Cmd + P`를 눌러 `Close`를 검색하여 "Close active pane"을 실행해 보세요.

## 2. 빠른 파일 이동 (`Cmd + O`)
파일 탐색기에서 마우스로 클릭하는 것보다 훨씬 빠릅니다.
- **실습**:
    1. `Cmd + O`를 누릅니다.
    2. `Overview`를 검색해서 프로젝트 오버뷰 파일로 이동해 보세요.

## 3. 링크 생성 (`[[`)
지식 연결의 핵심입니다.
- **문법**: `[[`를 입력하면 자동으로 파일 목록이 뜹니다.
- **별칭(Alias)**: `[[FileName|DisplayedName]]` 형식으로 파일명과 다른 텍스트를 보여줄 수 있습니다.
- **실습**: 아래에 대괄호 두 개를 입력하고 `Obsidian_Structure_Guide`를 링크해 보세요.
    - [ ] 링크 만들기: 

## 4. 링크 열기
- **이동**: 커서를 링크에 두고 `Cmd + Enter` (또는 `Option + Enter`로 새 창에서 열기).
- **미리보기**: `Cmd`를 누른 채 마우스를 링크 위에 올리면 팝업으로 내용을 볼 수 있습니다.

## 💡 Key Insights
*   **몰입 유지**: 단축키 사용의 진짜 목적은 0.1초의 속도가 아니라, 키보드와 마우스를 오가는 인지적 마찰을 줄여 '몰입(Flow)'을 유지하는 데 있습니다.
*   **기능의 접근성**: `Cmd + P` 팔레트는 수백 개의 기능을 외우지 않아도 즉시 접근하게 해주는 '만능 열쇠'입니다.
