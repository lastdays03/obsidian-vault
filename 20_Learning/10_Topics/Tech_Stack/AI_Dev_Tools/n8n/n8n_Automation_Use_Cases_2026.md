## 📖 정의 (Definition)
*n8n은 노드 기반의 워크플로우 자동화 도구로, 2026년 기준 AI 노드(OpenAI, Claude 등)와의 강력한 연동을 통해 단순 반복 업무를 넘어 복잡한 의사결정이 필요한 실무 영역까지 자동화 범위를 확장한 "AI Native Automation" 플랫폼이다.*

### 추천 공식 템플릿 (Starter Pack)

1.  **Gmail + OpenAI**: 이메일 자동 분류 및 중요 메일 슬랙 알림. (입문 추천)
2.  **Stripe + GPT-4o-mini**: 미납 청구서 리마인더 메시지 작성 및 발송.
3.  **Onboarding Task Generator**: 고객/직원 등록 시 체크리스트 생성 및 Notion 등록.
4.  **Legal Contract Analysis**: 계약서 PDF 위험 조항 추출 및 요약.
5.  **Notion Reviewer**: 특정 태그 페이지를 AI로 리뷰하고 피드백 남기기.
6.  **Report Generator**: 시트 데이터 요약 후 슬랙/이메일 리포트 발송.

## 💡 Key Insights
- **ROI 최적화 순서**: 체감 효과가 빠른 **Notion/DB 정리(4) → 보고서 자동화(2) → 리드 수집(1)** 순으로 도입하는 것이 조직 안착에 유리하다.
- **AI Agent로의 진화**: 단순 트리거-액션 방식(Zapier 등)을 넘어, AI가 판단하고 분기하는 **지능형 에이전트 워크플로우** 구현이 대세가 되었다.
- **진입 전략**: "Gmail + OpenAI + Slack" 조합으로 1시간 내에 "성공 경험"을 만드는 것이 확장의 핵심이다.
