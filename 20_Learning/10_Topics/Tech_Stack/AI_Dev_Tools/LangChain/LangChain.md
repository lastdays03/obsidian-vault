# LangChain

## 1. 개요 (Definition)
**LangChain**은 대규모 언어 모델(LLM) 기반 애플리케이션 개발을 위한 표준적인 프레임워크입니다.
복잡한 AI 워크플로우를 모듈식으로 연결하여 챗봇, 요약 시스템, 자율 에이전트 등을 효율적으로 구축할 수 있도록 돕습니다. 2026년 기준, LLM 앱 개발의 '표준(Standard)'으로 자리 잡았습니다.

## 2. 5가지 핵심 구성 요소 (Core Components)

LangChain은 다음 5가지 요소들을 유기적으로 조합할 때 진정한 가치를 발휘합니다.

1.  **모델 (Models)**:
    *   OpenAI, Anthropic, Google 등 다양한 LLM과 상호작용하기 위한 **통합 인터페이스**를 제공합니다.
    *   코드 수정 없이 모델을 쉽게 교체(Switching)할 수 있는 유연성을 보장합니다.
2.  **프롬프트 (Prompts)**:
    *   **PromptTemplate**을 사용하여 동적인 변수(Variables)를 포함한 프롬프트를 효율적으로 관리하고 재사용합니다.
3.  **체인 (Chains)**:
    *   여러 작업을 하나의 흐름으로 연결하는 **기본 단위**입니다.
    *   예: `User Input` → `Retrieval` → `Prompt` → `LLM` → `Output`.
4.  **메모리 (Memory)**:
    *   이전 대화 맥락(Context)을 저장하여 **멀티턴(Multi-turn) 대화**를 가능하게 합니다.
    *   단순 저장부터 요약 저장(Summary Buffer)까지 다양한 전략을 지원합니다.
5.  **에이전트 (Agents)**:
    *   정해진 절차(Chain)가 아니라, **LLM이 스스로 판단**하여 도구(Tool)를 선택하고 실행 순서를 결정하는 지능형 조정자입니다.

## 3. 장단점 분석 (Pros & Cons)

| 구분            | 주요 내용                                                                                                                                                                                                                         |
| :-------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **장점 (Pros)** | **압도적인 통합성**: 거의 모든 LLM, Vector DB, API와 연결 가능.<br>**개발 속도**: 사전 구축된 템플릿으로 빠른 프로토타이핑.<br>**유연성**: 프로세스 각 단계를 세밀하게 제어 가능.                                                 |
| **단점 (Cons)** | **추상화 복잡도**: 내부 동작을 이해하기 위한 초기 학습 곡선 존재.<br>**디버깅 난이도**: 긴 체인(Long Chain)의 경우 시각화 없이 문제 파악이 어려움.<br>**오버헤드**: 대규모 데이터 처리 시 순수 API 호출 대비 연산 효율 저하 가능. |

## 4. 2026년 최신 트렌드 (Trends)

*   **Agentic AI**: 단순 보조를 넘어 스스로 기획/실행하는 **자율 에이전트**로 진화.
*   **Multi-Agent**: 여러 전문 에이전트가 공유 상태(Shared State)를 업데이트하며 협업하는 구조 표준화.
*   **On-device**: 클라우드 의존도를 줄이고 개인정보 보호를 위한 **온디바이스 AI**와의 결합 확산.
