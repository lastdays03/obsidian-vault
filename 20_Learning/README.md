# 🧠 20_Learning (Knowledge Base)

**Role**: 지식소 (Archive 겸용).

프로젝트와 무관하게 평생 관리할 지식, 기술, 개념을 정리합니다.
특정 프로젝트에서 배우게 된 기술이라도, 범용적인 지식이라면 이곳에 정리하고 프로젝트에서 링크(`[[Link]]`)를 거는 것이 좋습니다.

## Categories
- `Tech_Stack/` : 언어, 프레임워크 (Python, React...)
- `CS_Concepts/` : 자료구조, 알고리즘, 디자인패턴
- `Domain/` : 산업 도메인 지식

## Tip
문서를 작성할 때 "이 지식이 어디에 쓰였는지"를 꼭 연결해두세요.
Example: `Related Project: [[Obsidian_MOC]]`
