# 📅 30_Journal (History)

**Role**: 나만의 역사 기록 (Chronological History).

시간의 흐름에 따른 기록을 남깁니다. 단순한 일기가 아니라, **"내가 오늘 무슨 프로젝트를 했고 무엇을 배웠는지"**를 연결하는 허브 역할을 합니다.

## Structure
- `Daily/YYYY-MM-DD`
- `Weekly/YYYY-Www`
- `Monthly/YYYY-MM`

## Daily Routine
1. 아침: 오늘 할 일(Projects) 계획.
2. 저녁: 오늘 한 일과 배운 점(Links to Learning) 회고.
